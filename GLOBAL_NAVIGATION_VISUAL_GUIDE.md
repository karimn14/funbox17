## Context-Sensitive Behavior

### E Button (Next/Enter) Context

| Screen | E Button Action |
|--------|----------------|
| Story | Advance to Videos or Activities or Quiz |
| Video (not last) | Play next video |
| Video (last) | Go to Activities or Quiz |
| Activity (not last) | Go to next activity |
| Activity (last) | Go to Quiz |
| Quiz | (Handled by quiz logic - not global nav) |
| Result | Return to Home |

### F Button (Back) Context

| Screen | F Button Action |
|--------|----------------|
| Story | Return to Home |
| Video (first) | Return to Story or Home |
| Video (not first) | Go to previous video |
| Activity (first) | Go to last Video or Story or Home |
| Activity (not first) | Go to previous activity |
| Quiz | Go to last Activity or Video or Story or Home |
| Result | Return to Quiz |

## State Management During Navigation

```
When pressing F to go back:
  1. Current step identified
  2. Previous step calculated
  3. State reset (feedback, selections cleared)
  4. Index adjusted (if multi-item step)
  5. Step changed
  6. New content rendered

When pressing E to go forward:
  1. Current step identified
  2. Next step calculated
  3. State reset (feedback, selections cleared)
  4. Index adjusted (if multi-item step)
  5. Step changed
  6. New content rendered
```
